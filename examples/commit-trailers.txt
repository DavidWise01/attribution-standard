# ROOT0 Attribution Standard — Git Commit Trailer Format
#
# Add these trailers to any commit message that includes AI contribution.
# Place them after the commit body, separated by a blank line.
# The Co-Authored-By trailer is parsed natively by GitHub and GitLab.
# The AI-* trailers are parsed by ROOT0-compatible tooling.
#
# ── EXAMPLE COMMIT ──────────────────────────────────────────────────────────

observer pattern. Lineage remains intact through all eight observer
states. Genesis equation holds.

#
# Co-Authored-By     Required. Parsed by GitHub/GitLab. Format:
#                    Name (Model) <email-or-noreply>
#

#                    intellect · generation · execution
#                    review · validation
#                    witness · observation
#

#
